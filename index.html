HTML

  • Every HTML starts off with a HEAD then followed by BODY
  • The HEAD element contains mostly metacontent, or data about the content
  • META elements tell things like the character set and tab title
  • TITLE element is the only visible part of the HEAD, and is on the tab of a browser
  • The BODY element contains the stuff you see on the website
  • HEADER elements are at the very top; whereas FOOTER elements are at the very end

Git

  • status: checks the current branch
  • checkout -b branch-name: create and move into branch-name
  • add -A: stages ALL files in preparation for commit
  • commit -m "message": saves changes on your branch locally
  • pull origin main: checks/updates local env from main branch
  • push origin branch-name: submits changes to branch to hub
